# MyVideoShop 短视频带货商城系统
基于 Uniapp + Webman 开发的高性能短视频电商系统 | 全开源无加密 | 支持短视频带货 + 分销代理

## 项目介绍
MyVideoShop 是一套专注于短视频带货的电商系统，采用 Uniapp 前端 + Webman 高性能后端开发，
支持短视频种草、商品购买、分销代理、活动营销等核心功能，后台简洁易用，前端美观大方。

系统功能完整，离商用仅一步之遥，适合开发者学习、二次开发、项目落地使用。

## 功能亮点
 - ✅ **短视频带货**：视频内直接挂载商品，一键跳转购买，转化路径短
 - ✅ **完整电商模块**：商品管理、订单管理、库存管理、支付对接
 - ✅ **分销代理系统**：推广分成、下级返利、佣金提现，快速裂变获客
 - ✅ **丰富营销活动**：转盘抽奖、砸金蛋、优惠券、限时折扣等
 - ✅ **Uniapp 全端支持**：一套代码，多端发布（APP/小程序/H5）
 - ✅ **Webman 高性能后端**：高并发、低延迟，稳定支撑大流量场景
 - ✅ **全开源无加密**：无后门、无云验证，可商用、可二次开发

## 技术栈
- 前端：Uniapp
- 后端：Webman (PHP 高性能框架)
- 数据库：MySQL
- 支持：Android / iOS / H5 / 小程序

## 适用场景
- 短视频带货平台创业
- 电商直播、内容电商项目
- 分销代理、私域流量变现
- 二开学习、外包项目交付
